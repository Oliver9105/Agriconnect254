# AgriConnect 2026

A high-performance agricultural supply chain platform connecting Kericho farmers with Nairobi markets, featuring NeuralAgri growth predictions and M-Pesa escrow.

## Author

**Oliver** — [github.com/Oliver9105](https://github.com/Oliver9105)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   ```bash
   npm run dev
   ```
